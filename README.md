# GLBViewer

> **Upload once. Share forever. In 3D.** Drag-and-drop .glb / .gltf / .obj files and get an instant shareable Three.js viewer link.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Features
- 📤 Drag-and-drop upload (.glb, .gltf, .obj)
- 🌍 Instant shareable URL — no install needed to view
- 🕯️ Orbit, zoom, pan controls
- 💡 HDRI environment lighting presets
- 🎥 Animation playback with timeline
- 🔤 Morph target / blend shape sliders
- 📱 Mobile-responsive viewer

## Tech Stack
`React` · `Three.js` · `React Three Fiber` · `Supabase Storage` · `TypeScript`
